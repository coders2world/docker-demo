# 📄 Project Description

## Title:

Kubernetes Deployment using Minikube

## Description:

This project focuses on deploying a containerized application using Kubernetes in a local environment with Minikube. The main objective is to understand how Kubernetes manages applications, scales them, and ensures high availability.

## Features:

* Deployment of application using YAML configuration
* Scaling pods dynamically
* Performing rolling updates
* Rolling back to previous stable version
* Monitoring using kubectl commands

## Tools Used:

* Kubernetes
* Minikube
* Docker
* kubectl

## Outcome:

Successfully deployed and managed an application in a Kubernetes cluster with real-world DevOps practices.

## Conclusion:

This project provides hands-on experience with Kubernetes fundamentals and prepares for real-world DevOps and cloud-based deployment scenarios.
